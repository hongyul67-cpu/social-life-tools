# 🏫 사회생활 준비 학습 도구 모음

사회생활(자립·진로) 준비 수업에 쓰는 학습 도구 모음입니다. 여러 활동을 카드로 모아
한 페이지에서 골라 쓸 수 있습니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/social-life-tools/

## 주요 기능
- 사회생활 준비 주제별 **학습 도구 모음** (카드형 메인에서 선택)
- 수업·자기주도 학습에 바로 활용

## 사용 방법
- 위 주소를 열면 도구 모음 메인이 열립니다. 원하는 활동 카드를 눌러 사용하세요.
- 설치가 필요 없으며 PC·태블릿·스마트폰에서 동작합니다.
